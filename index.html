Best FREE Resources

Course: TryHackMe - Linux Fundamentals (Free)

Common Mistakes Using sudo everywhere, ignoring hidden files (ls -la), forgetting absolute vs relative paths.

Essential Tool Arsenal

Every tool below is free. Installation guides assume a fresh Ubuntu or Kali Linux VM. All explanations include why, when, and how with examples.

Kali Linux in VirtualBox / VMware
RECOMMENDED SETUP

Why: Pre-installed with 600+ security tools. Isolated environment for safe practice.

How: Download VirtualBox (free) → Download Kali Linux ISO or pre-built VM from kali.org → Import and run.

# Update system
sudo apt update && sudo apt full-upgrade -y

Alternative: WSL2 on Windows (good for beginners) or Docker (advanced).

Burp Suite Community
WEB PROXY & SCANNER

When: Intercepting and modifying HTTP requests (Phase 4 Web).

How: Download from portswigger.net → Run JAR or use installer. Set browser proxy to 127.0.0.1:8080. Use Repeater, Intruder, Decoder tabs.

Right-click request → Send to Repeater. Modify and send to test for vulnerabilities.
Nmap
NETWORK SCANNER

Why: Discover hosts, open ports, services, OS fingerprinting.

nmap -sV -sC target.com
nmap -p- --min-rate 1000 target.com

Cheat sheet available in Cheat Sheets tab.
