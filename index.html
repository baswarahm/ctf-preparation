Learning Roadmap

WEEK 1
3 – 9 JULY
Linux Fundamentals
Tasks
  • • Complete OverTheWire Bandit levels 0–20
  • • Explore easy challenges on PicoCTF
Goal

Become comfortable using the terminal without a GUI.

WEEK 2
10 – 16 JULY
Networking + Intro Session
Tasks
  • • Watch free Networking videos
  • • Practice basic nmap
  • • Attend official Introduction Session (11 July)
Goal

Understand ports, services, and basic scanning.

WEEKS 3–4
17 JUL – 30 JUL
Web Exploitation
Tasks
  • • Complete labs on PortSwigger Web Security Academy
  • • Solve Web challenges on PicoCTF
Goal

Master common web vulnerabilities.

WEEK 5
31 JUL – 6 AUG
Cryptography + Hacker Class
Tasks
  • • Complete CryptoHack Beginner path
  • • Attend free Hacker Class
Goal

Understand encoding, hashing, and basic encryption.

WEEK 6
7 – 13 AUG
Forensics & Steganography
Tasks
  • • Solve Forensics challenges on PicoCTF
  • • Attend Live Hacker Class (9 Aug)
Goal

Extract hidden data from files and images.

WEEK 7
14 – 20 AUG
Reverse Engineering & Binary
Tasks
  • • Start pwn.college baby modules
  • • Attend Live Hacker Class (16 Aug)
Goal

Decompile simple binaries and find flags.

FINAL WEEK
21 – 29 AUG
Full Practice & CTF Simulation
Tasks
  • • Solve full CTF sets on PicoCTF
  • • Practice on CTFlearn.com
  • • Team practice + rest on 28 Aug
Goal

Be fully prepared for the Preliminary Round.
